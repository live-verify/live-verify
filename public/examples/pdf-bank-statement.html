PDF Bank Statement Verification

Verifying a claim inside a PDF document via text selection

What is this?

This page renders a mock bank statement PDF using PDF.js. The PDF contains a verify: line at the bottom. You can select the text in the rendered PDF (including the verify line) and verify it — just as you would on any web page.

This demonstrates that PDF documents with verification lines can participate in the Live Verify ecosystem. A native PDF viewer (Preview, Evince, Adobe Reader) with text-selection verification built in would provide the same experience.

A small leap of faith

Ideally we'd demonstrate this inside a native PDF viewer (Qt/GTK+/Cocoa) with built-in verification. Instead, we use PDF.js to render the PDF in-browser with a selectable text layer — the same text layer that native viewers provide. The browser extension's text-selection verification works identically on this rendered text. Take the leap: if it works here, it works in any viewer that exposes selectable text.
